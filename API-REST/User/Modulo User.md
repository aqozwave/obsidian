## Associate
Este modulo se conectara con [[Modulo Auth]] tendra asociado una [[Modulo Business]] [[Modulo Ubication]]
Un [[Modulo User]] puede estar asociado a una [[Modulo Business]]
Un [[Modulo User]] puede estar asociado a una [[Modulo Ubication]]
Un [[Modulo User]] tendra asociado un [[Modulo CrewMember]] si es que es necesario atraves de [[RUN]]
Un [[Modulo User]] tendra asociado un [[Modulo Academic_degrees]]
Un [[Modulo User]] tendra un [[ROL]] . Ejemplo : Admin 
Un [[Modulo User]] tendra asociada un [[Modulo Area]] donde este trabaja por ejemplo : Puente
Tambien validara si el [[RUN]] esta asociado algun [[Modulo CrewMember]]
