	API REST que gestiona cursos y certificaciones obligatorias de tripulantes de embarcaciones, con control de usuarios, roles y notificaciones de vencimientos, orientada al cumplimiento de la normativa marítima en Chile.
	
> [!attention] Multitenant > Es un sistema multitenant donde múltiples empresas > comparten la misma instancia. Cada empresa tiene sus > propias ubicaciones y áreas, y los datos están > completamente aislados entre tenants.

---
# 🎯 Propósito de la aplicación

### **Qué es:**

	Una plataforma backend y frontend para centralizar la información de cursos obligatorios de tripulantes.
	
	**Para qué sirve:**
	
	Mantener la documentación al día, evitar vencimientos y facilitar la presentación ante la autoridad marítima.
	
	**Quién la usa:**
	
	- Tripulantes
	    
	- Administradores de flota / empresas marítimas
	    
	- Personal encargado de cumplimiento normativo


---

## 🗄️ Entidades principales

	- Usuario
	- Rol
	- Tripulante
	- Curso   
	- CursoTripulante
	- Notificación
